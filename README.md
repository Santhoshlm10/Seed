**Seed** is an open-source, fully offline mock data generator extension designed for developers, QA engineers, and designers. Generate millions of rows of realistic dummy data tailored to your needs, or instantly inject random data into your active browsing tab without typing a thing.

## Features

- **100% Offline & Private:** No API calls, no servers. Everything is generated safely via Web Workers locally on your machine.
- **Instant Form Injection:** Stop typing `test1@example.com` a hundred times a day. Map website form inputs by ID, CSS selector, or XPath, and auto-magically fill forms on any page.
- **Data Generation & Export:** Create huge JSON arrays of deeply customizable data types ranging from emails, credit card numbers, fake addresses, to complex regex-based strings.
- **Import / Export Schema & Forms:** Save your data schemas or form injection mappings as JSON and share them with your team to sync QA processes.
- **Native Light/Dark Mode:** A sleek UI that matches your system preferences.

## How it Works

Seed is split into two primary modules:

| Module | Description |
|---|---|
| **Generator** | Define your column names and pick from hundreds of `faker.js` data formats. Export massive datasets to your local drive instantly. |
| **Form Injector** | Tell Seed what inputs exist on a target webpage (e.g., `#email`, `#password`), assign a data type to them, and hit **Inject**. Seed takes care of firing the correct React/Vue bindings to ensure the webpage registers the input logic. |

## Installation (Developer Mode)

To run the extension locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Santhoshlm10/seed.git
   cd seed
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the extension:
   ```bash
   npm run build
   ```
4. Load into Chrome:
   - Go to `chrome://extensions/`
   - Enable **Developer mode** in the top right.
   - Click **Load unpacked**.
   - Select the `dist` folder generated by the build step.
5. Click the Seed puzzle piece in your extensions bar!
